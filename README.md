# Event Planner Monorepo

This repository is now organized into two folders:

- `backend/` → existing Express + TypeScript API
- `frontend/` → Vite + React + TypeScript client scaffold

## Quick start

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Set `frontend/.env` using `frontend/.env.example` so the frontend can call the API.
